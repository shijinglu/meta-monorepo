# meta-monorepo
a mono repo based on git-meta

```
$ git submodule add -b master git@github.com:xueqilsj/xueqilsj.github.io.git
$ git submodule add -b master git@github.com:shijinglu/route.git
$ alias gm='~/workspace/git-meta/node/bin/git-meta'
$ gm add .gm commit -m 'first monorepo commit'
$ gm status
>  On branch [32mmaster[39m.
>  nothing to commit, working tree clean


```
